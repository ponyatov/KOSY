Der Aufbau eines Zeichens

Die mit dem ZSE erstellbaren Zeichensätze sind sogenannte Vektorzeichensätze. Das heißt, ein Zeichen besteht aus einer Anzahl Linien, die von der NC-Maschine zum Gravieren oder Fräsen abgefahren werden. Der Vorteil der Verktorzeichensätze besteht darin, dass sie ohne Qualitätsverlust vergrößert oder gedreht werden können.
Jede Linie wird durch einen Start- und einen Endpunkt charakterisiert. Schließt sich eine Line direkt an eine andere Linie an (Polygon), teilen sich beide Linien den End- bzw. Startpunkt.
Jedes Zeichen im QFN-Zeichensatz kann aus Linien mit bis zu 220 Stützpunkten bestehen. Zum Zeichnen von Linien steht ein Raster von 64 x 64 diskreten Koordinatenpunkten zur Verfügung.
