Die Menüfunktionen

Menü Werkzeuge


Bild: Menü Werkzeuge

 Im Menü Werkzeuge kann über "Neu darstellen" eine Auffrischung des Rasterzeichenfeldes erzwungen werden.


Bild: Dialog Schriftart Vorlage

 Im Dialog "Schriftart Vorlage" wird der Hintergrundzeichensatz ausgewählt. Dieser kann zur Erleichterung und Orientierung beim Entwurf eines Vektorzeichensatzes im Hintergrund des Zeichenrasters eingeblendet werden. Dazu ist jeder auf Ihrem PC installierte Zeichensatz geeignet. Im Dialog können Schriftart, Form und Farbe gewählt werden. Die eingestellte Größenangabe entspricht der Rastereinheit. Sie können den Knopf "Apply" nutzen, um die Schrift im Zeichenraster zu probieren ohne den Dialog zu verlassen. Die eingestellte Schrift wird relativ zur Basislinie mit der in der Werkzeugpalette vorgegebenen Position eingeblendet.


Bild: Dialog Schriftprobe

 Im Dialog "Schriftprobe" kann der Vektorzeichensatz für die Gravur geprobt werden. Dabei wird ein frei wählbarer Probetext so dargestellt, wie er später graviert wird. Schriftneigung, Zeichengröße und -abstand sind einstellbar.


Bild: Dialog Outlinetest Zeichensatz

 Mit "Outlinetest Zeichen / Zeichensatz" kann geprüft werden, ob ein Zeichen oder der Zeichensatz vollständig aus geschlossenen Polygonzügen besteht. Dies sollte z.B. bei Outlinezeichensätzen der Fall sein. Mit diesem Werkzeug kann sichergestellt werden, dass keine augenscheinlich geschlossene Zeichen die jedoch Linienüberlappungen aufweisen enthalten sind. So werden Fehler beim Füllen oder Flächenausfräsen vermieden. Die Start- und Endposition einer Linie die nicht an andere Linien anschließt wird zur Fehlerlokalisierung aufgelistet. Um in das fehlerhafte Zeichen zu gelangen, reicht ein Klick auf die entsprechende Zeile in der Fehlerliste.
