Menü Datei
Überarbeitet: Oktober 2004

Am Anfang jeder Programmier-Tätigkeit mit nccad steht die Auswahl einer Datei, die bearbeitet werden soll. Entweder es wird eine neue Datei begonnen, oder eine vorhandene zur Weiterbearbeitung aufgerufen. Außerdem werden über dieses Menü die Funktionen "Speichern" und "Drucken" gestartet. Der ordnungsgemäße Ausstieg aus nccad wird ebenfalls über dieses Menü erreicht.

Grundsätzlich wird im Menü Datei die Entscheidung getroffen, ob die Programmierung der CNC-Maschine nach der CNC-Methode oder nach der CAD/CAM-Methode erfolgen soll.

Im Einzelnen ist Folgendes zu beachten:

  • CAD/CAM - Neue Zeichnung
    Der Weg von der Zeichnung zum Werkstück beginnt hier.
    Für den Fall, daß eine Datei bereits in Arbeit ist, wird zunächst gefragt, ob sie abgespeichert werden soll. Danach wird der Bildschirm erneuert und ein leeres Zeichenfeld in Form des Y-Tisches dargestellt. Gleichzeitig wird der Speicher für die Zeichnungsteile gelöscht. Die eigentlichen Zeichenfunktionen wählen Sie über die Iconleiste.

  • CAD/CAM - Neue Spezialzeichnung
    Hier sind die nicht alltäglichen CAD-Funktionen zusammengefaßt. Es erscheint ein weiteres Untermenü, in dem gewählt werden muß.

    • Drehen
      Wenn Sie über den Drehzusatz verfügen, werden die Software-Funktionen zugänglich.
      Die Bedienung wird im zugehörigen Handbuch bzw. im Hilfethema erläutert.
    • Modellbau
      Speziell Modellbauer können mit den zugehörigen Funktionen Rippenkörper konstruieren (Flügel, Rümpfe u.A.m.)
      Die Bedienung wird im Hilfethema 3D/Modellbau erläutert.
  • CAD/CAM - Zeichnung öffnen
    Eine vorhandene Zeichnung soll geladen werden. Für den Fall, daß eine Datei bereits in Arbeit ist, wird zunächst gefragt, ob sie abgespeichert werden soll. Danach wird der Bildschirm erneuert und ein Auswahlfenster mit den, im angemeldeten Verzeichnis vorhandenen, CAD-Dateien geöffnet. Wählen Sie die gewünschte Datei.
  • CNC - Neues Programm
    Ein neues NC-Programm soll begonnen werden. Für den Fall, daß eine Datei bereits in Arbeit ist, wird zunächst gefragt, ob sie abgespeichert werden soll. Danach wird der Bildschirm erneuert und ein leeres Editierfeld dargestellt. Gleichzeitig wird der Speicher für die NC-Sätze gelöscht.
  • CNC - Programm öffnen
    Über das Explorer-Fenster wird eine KOSY-CNC-Datei mit der Extension *.KNC ausgewählt und geladen. Gleichzeitig wird der Texteditor geöffnet.
    Ein vorhandenes NC-Programm soll geladen werden. Für den Fall, daß eine Datei bereits in Arbeit ist, wird zunächst gefragt, ob sie abgespeichert werden soll. Danach wird der Bildschirm erneuert und ein Auswahlfenster mit den, im angemeldeten Verzeichnis vorhandenen, CNC-Programmen geöffnet. Wählen Sie die gewünschte Datei aus.
  • Datei importieren
    Hier sind die nicht alltäglichen Lade-Funktionen zusammengefaßt. Es erscheint ein weiteres Untermenü, in dem gewählt werden muß.
    • DXF
      Sind Zeichnungen mit anderen CAD Programmen erstellt- und als DXF-Dateien abgespeichert worden, können sie hier eingelesen werden.
      Die Ergänzung mit Technologiedaten wird dann in nccad vorgenommen.
      Fremde Zeichnungen werden von Layer 0 bis 32000 erfaßt, Layer 0 wird auf Layer 1 umgesetzt und Layer größer 9 erscheinen in den Layern 1 bis 9. Farben werden beim Import nur im Zusammenhang mit Punkten ausgewertet. Zu große Zeichnungen werden beim Import automatisch mit einem ganzzahligen Maßstabsfaktor auf Tischmaß verkleinert. Bei zu kleinen Zeichnungen wird die Frage gestellt, ob der Maßstab angepaßt werden soll oder nicht, erst dann wird eingelesen und dabei entweder die Zeichnung belassen oder auf Tischmaße vergrößert. Die Abarbeitung der im Maßstab veränderten Zeichnungen mit Grafiksimulation oder mit der CNC-Maschine erfolgt allerdings in Orginalgröße, wenn Sie nicht im Menü <Parameter> einen neuen Maßstabswert angeben.        Siehe auch unter Import/Export 2D .
    • HPGL
      Manche CAD-Programme liefern sog. Plotter-Dateien (HPGL-Dateien). Wenn die CNC-Maschine den Plotter ersetzen soll, ist diese Importmöglichkeit von Vorteil.
      Die Technologie-Angaben von nccad bieten zudem die Möglichkeit der Bahnkorrektur und Mehrfach-Zustellung.
      Siehe auch unter Import/Export 2D .
    • Oberflächen-Datei
      Wurde zuvor ein plastischer Körper abgetastet und das Ergebnis abgespeichert (*.txt), kann es über diese Funktion wieder geladen werden. Auf der Zeichenfläche erscheint dann eine Plastische Zone.
    • STL-Datei
      Werden 3D-Körper mit fremder CAD gezeichnet und als STL-Datei abgespeichert (*.stl), dann können sie hier importiert werden. Alles Weitere siehe unter Hilfethemen CAD 3D/STL.
    • Bild-Datei
      Hier könne Bild-Dateien in den Formaten BMP, GIF und JPG importiert werden. Mehr dazu erfahren Sie unter Grundlagen CAD/Grafik......
  • Letzte Datei(en)
    Sie erhalten eine Auflistung der letzten, von Ihnen bearbeiteten, Dateien, für einen schnellen Zugriff.
  • Datei speichern
    Die gerade in Arbeit befindliche Datei wird im angemeldeten Verzeichnis gespeichert. Sollte noch kein Name angegeben worden sein (in der Menüzeile oben steht unter Dateiname "KeinName" ) wird zunächst das Datum und der Name des Bearbeiters gefordert. Erst danach wird das Explorer-Fenster für die Eingabe des Dateinamens geöffnet.
  • Datei speichern unter
    Es wird ein Fenster geöffnet, in dem Datum und Name des Bearbeiters einzugeben sind. Erst danach wird das Explorer-Fenster für die Eingabe des Dateinamens geöffnet.
    Auf diese Weise lassen sich vorhandene Dateien unter neuem Namen abspeichern, wenn sie z.B. abgeändert wurden. Die Datei wird unter dem angegebenen Namen im angemeldeten Verzeichnis abgespeichert. Das Verzeichnis kann aber auch verändert werden.
  • Datei exportieren
    Hier sind die nicht alltäglichen Speicher Funktionen zusammengefaßt. Es erscheint ein weiteres Untermenü, in dem gewählt werden muß.
    • Modellbau (3D ->2D)
      Sind für Rippenkonstruktionen unter "3D/Modellbau" mehrere Rippenprofile entstanden, werden sie durch diesen Export-Aufruf auf eine Fläche projeziert und können als 2D Datei weiter bearbeitet werden (z.B. Verteilen, mit Technologie versorgen und Ausfräsen).
    • HPGL Datei
      Die Zeichnung wird als Plott-Datei (*.PLT) abgespeichert.
      Sie kann dann entweder einem Plotter zugeführt werden oder in ein DTP-Programm importiert werden.Wenn Sie Zeichnungen zum HPGL-Export machen, sollten Sie keine True-Type-Schriften einbinden (Beschriftung, Gravurtexte), sie können nicht dargestellt- oder ausgeplottet werden.
    • KOSY Liste
      Wurde eine Zeichnung mit Technologiedaten erstellt, dann wird üblicherweise im Menü <Maschine> ein Bearbeitungsablauf gestartet , oder die Grafik-Simulation aufgerufen. In jedem Fall wird zunächst der Postprozessor aktiviert, der eine Steuerliste für die CNC-Maschine erzeugt. Diese Steuerliste kann separat abgespeichert werden, sie hat denselben Namen wie die Zeichnungs-Datei, jedoch die Erweiterung *.CNC.
  • Seite einrichten
    Alle Vorbereitungen zum Drucken werden hier getroffen. Das zugehörige Fenster ist sehr übersichtlich und erklärt sich selbst. Weitere Hinweise finden Sie unter Grundlagen der Bedienung/Drucken...
  • Datei drucken
    Die aktuelle Datei wird ausgedruckt.
    Die Druckereinstellung finden Sie "Seite einrichten" (siehe oben).
  • Konfiguration
    Hierzu finden Sie ausführliche Informationen unter Grundlagen der Bedienung/Konfiguration....
  • Vorlagen
    Hierzu finden Sie ausführliche Informationen unter Grundlagen der Bedienung/Vorlagen....
  • nccad6 Ende
    Der ordnungsgemäße Ausstieg aus nccad über diese Menü-Funktion ist immer zu empfehlen, es werden alle Einstellungen abgespeichert.
