nccad7 in Windows-Netzwerken
Stand: Februar 2005

Grundsätzlich kann jede Software in Netzwerken irgendwie installiert werden. Besser ist es aber, wenn der Software-Hersteller die Anwendung seiner Software im Netzwerk unterstützt, falls entsprechende Bedürfnisse vorhanden sind.

Ab der Version 7 von nccad (ab Oktober 2004) sind folgende netzwerkbezogene Erweiterungen verfügbar:

  • Installationshilfe
    Ein sog. MSI-Paket ermöglicht die Automatisierung der Installation in Windows-Netzwerken. Als Client-Betriebssystem wird Windows2000 Professional genutzt.
  • Arbeiten im Netz
    Individuelle Einstellungen, Speicherorte, Arbeitsweisen müssen im Netz bedienerfreundlich organisiert- und verwaltet werden.

dazu die folgenden, weiterführenden Hinweise:

1. Installationshilfe unter MS-Windows
Zum komfortablen Installieren ist auf der glieferten CD unter Installation\Deutsch im Unter-Ordner 04_Netzwerk ein spezielles MSI-Paket verfügber:

  • Für Netzwerk-Installation
    • Das Programm nccad7.msi
      Nach der Installation sind drei Start-Icon am Bildschirm vorhanden, diese entsprechen der Installationsversion "Ausbildung".
      Es ist am Server zusammen mit der Netzwerk-Verwaltungs-Software (z.B. ActiveDirectory) zu verwenden.

1.1 Musterlösung für Netzwerk-Installationen
Es gibt Netzwerke, die nach einer sog. Musterlösung eingerichtet werden müssen. Das oben genannte ***.msi -Programme muß dem Netzwerk-Administrator übergeben werden, der die Installation von nccad entsprechend der Anleitung für die Musterlösung vornimmt.

Bei älteren Windows-Versionen (Win98 usw.) muß evtl. ein neuer Windows-Installer (Version2) nachinstalliert werden. Dies sollte aus den auftretenden Fehlermeldungen ersichtlich sein.

Zu beachten ist, dass für die Installation von Programmen im Regelfall Administrator-Rechte erforderlich sind.

Für die Ausführung des Programms nccad7 sind Schreib-/ Leserechte in der Datei nccad45.mdb erforderlich. Diese wird standardmäßig ins Verzeichnis C:\kosy7\ installiert, kann aber auch anschließend in ein separates Verzeichnis nach Wahl verschoben werden.
Natürlich müssen die Zugriffsrechte für diese Datei angepaßt werden, egal wo sie abgelegt wurde.
Ist ein anderes Verzeichnis als C:\kosy7\ gewählt, muß in der Datei nccad7.ini innerhalb der Parameter-Gruppe [Administrator] der spezielle Parameter DatabasePath= ******** angepasst werden.

Bei abweichenden Client-Betriebssystemen sowie notwendigen oder gewünschten Anpassungen hinsichtlich Programm-Start und -Konfiguration kann mit geeigneter Software ein eigenes MSI-Paket erstellt werden. Hinweise dazu finden Sie In der Dokumentation zum Netzwerk.

Der Aufruf einer vorhandenen Konfigurationsdatei mittels Desktop-Verknüpfung erfolgt durch Parameterübergabe. Dazu markieren Sie die Desktop-Verknüpfung, rufen mit Rechtsklick das Kontextmenü auf und wählen den Punkt Eigenschaften. Im Feld Ziel: geben Sie nach dem Pfad zur Programmdatei den Parameteraufruf mit Pfad zur gewünschten Konfigurations-Datei an (zum Beispiel: C:\kosy7\nccad7.exe /men:C:\kosy7\210_Fortgeschrittener.ncc).

2. Arbeiten im Netz
Eine Reihe von wesentlichen Veränderungen gegenüber nccad6 und älter führt zu einem komfortableren Arbeiten im Netz und erweitert die Einsatzmöglichkeiten:

2.1 Die Datei nccad*.ini
In allen bishereigen Versionen gab es nur eine Datei und eine Speicherstelle für die individuellen- und maschinenbezogenen Einstellungen. Ab nccad7 gibt es 2 verschiedene Dateien, die beim ersten Programmstart erstellt werden:

Die Datei nccad7.ini
Sie wird nun (Windows-Version-abhängig) meist im Verzeichnis...
C:\Dokumente und Einstellungen\All Users\Anwendungsdaten\kosy\kosy7
 ...gespeichert.
Sie enthält sämtliche Maschineneinstellungen und ist für alle Benutzer gleich.

Die Datei nccaduser.ini
Sie wird nun (Windows-Version-abhängig) meist im Verzeichnis...
C:\Dokumente und Einstellungen\Benutzername\Anwendungsdaten\kosy\kosy7
... gespeichert.
Sie enthält alle benutzerspezifischen Einstellungen. In einem normalen Netzwerk wird dieses
Verzeichnis bei Anmeldung des Benutzers immer aus dem Nutzerverzeichnis des Server geladen
und auch dahin gesichert. Somit kann jeder Benutzer eigene Einstellungen vornehmen und hat sie
an jedem Rechner nach der Anmeldung parat.
Hinweis: Ein automatischer Import der Einstellungen aus älteren nccad-Versionen ist eingebaut.

2.2 Vorlagen-Dateien ****.cdt
Immer wenn in einem Team gearbeitet wird, müssen gewisse Standards eingehalten werden. Vorlagen, die für alle zugänglich sind, können helfen.

Im Menü Datei/Vorlagen finden Sie deshalb eine neue Gruppe für folgende Funktionen:

  • Als Vorlage speichern
  • Vorlage öffnen
  • Vorlage bearbeiten

Vorlagen enthalten im wesentlichen Basis-Zeichnungen. Will man sie verändern, so muß man die Vorlage erst
über den Menüpunkt Vorlagen/Vorlage bearbeiten freischalten.
