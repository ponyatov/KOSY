Schaltfunktionen während einer linearen Bewegung
Überarbeitet: September 2004

Bitte beachten:
Diese Funktion erfordert einen speziellen Microcontroller, bitte fragen Sie beim Hersteller danach.


1. Beispielzeichnung mit 4 Zusatztechnologien
Mit der Zusatztechnologie können zusätzliche Funktionen (Tiefenregelung/Relais schalten) während einer Bewegung an bestimmten Positionen ausgeführt werden. Zusatztechnologien können z.Zt. nur an Geraden vergeben werden. Ferner können Zusatztechnologien z.Zt. nicht an das Ende einer Geraden angehängt werden.


2. Das Icon "ZUSATZTECHNOLOGIE"
Sie erreichen das Icon "Zusatztechnologie" über das Icon "TECHNOLOGIE-SONSTIGE" in der Haupticonleiste. Nachdem Sie das Icon angeklickt haben, müssen Sie den Startpunkt der Zuordnungslinie festlegen. An dieser Position werden später die angegebenen Funktionen ausgeführt. Bei horizontalen Geraden ist zu beachten, dass der Y-Wert des Startpunktes mit dem Y-Wert der Geraden exakt übereinstimmt. Bei vertikalen Geraden müssen entsprechend die X-Werte übereinstimmen, ggf. sollten die Koordinaten manuell eingegeben werden. Nachdem Sie den Endpunkt der Zuordnungslinie gewählt haben erscheint das Zusatztechnologie-Fenster


3. Das Zusatztechnologie-Fenster
Zunächst geben Sie unter "Gehört zu Technologie Nummer" die Nummer der Technologie an, zu der diese Zusatztechnologie gehört. Versehen Sie das Kontrollkästchen "Tiefenregelung" - durch Anklicken - mit einem Kreuzchen, wird  die Tiefenregelung aktiviert. Andernfalls wird die Tiefenregelung deaktivert, die Bewegung wird dann mit der zuletzt erreichten Z-Höhe fortgesetzt. Bei "Relais an Position" kann der Zustand der Relais eingestellt werden, der bei Erreichen der angegebenen Position hergestellt werden soll. Ein Kreuzchen in einem der Kontrollkästchen schaltet das entsprechende Relais ein, fehlt das Kreuzchen, wird das entsprechende Relais ausgeschaltet. Unter "Wartezeit" kann eine Zeit eingestellt werden, die verstreichen muss, bevor der - unter "Relais nach Wartezeit" eingestellte - Relaiszustand hergestellt wird. Der Wert, der hier eingestellt werden kann, wird nicht in Sekunden angegeben. Vielmehr stellt er ein ganzzahliges Vielfaches von 55 ms dar, d.h. ein Wert von 18 entspricht ca. 1 Sekunde. Während der "Wartezeit" bewegt sich die Maschine natürlich weiter. Ist die Bewegung beendet bevor die  eingestellte Zeit abgelaufen ist, wartet die Maschine bis zum Ablauf der Zeit und führt dann die Schaltfunktionen aus. Unter "Relais nach Wartezeit" kann der Relaiszustand nach Ablauf der "Wartezeit" eingestellt werden.

Zusatztechnologien können nachträglich bearbeitet werden in dem man wieder das Icon "ZUSATZTECHNOLOGIE" anwählt. Allerdings muss dann der "Konstruktionsfang" - über die Tastenkombination [Strg]+[F9] - ggf. eingeschaltet werden, damit das Ende der Zuordnungslinie gefangen werden kann. Alternativ kann man aber auch das Icon "TECHNOLOGIE" anwählen, um eine Zusatztechnologie zu bearbeiten.
