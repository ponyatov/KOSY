Zeichnungsteil-Information

Icon in der Icon-Gruppe "Information"
Überarbeitet: August 2002

Ein Zeichnungsteil ist mit dem Suchfenster auszuwählen, dann erhalten Sie alle erreichbaren - und berechenbaren Informationen zu diesem Teil.

Nach einem Klick auf das Icon sind folgende Tätigkeiten erforderlich:

  1. Fadenkreuz mit dem Suchfenster an einem Konstruktionspunkt positionieren und auf ein "Einrasten" warten (Teil wird farbig markiert).
  2. Statuszeile lesen. Sie gibt eine Bestätigung, daß das gewünschte Teil gefunden wurde.
  3. Mit einem Mausklick das gefundene Teil bestätigen - oder durch Klick auf ein anderes Icon die Funktion abbrechen.

Das Informations-Fenster liefert...

... im Rahmen "Allgemein" die direkt beim Zeichnen entstandenen Daten,

... im Rahmen "Informationen zum ..." errechenbare Daten, wie z.B. Flächeninhalt,

... in der Tabelle eine Liste von vorhandenen Konstruktionspunkten (falls es sich um zusammengesetzte Zeichnungsteile handelt).
