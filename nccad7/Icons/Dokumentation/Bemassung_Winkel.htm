Winkel-Bemaßung

Icon in der Icon-Gruppe "Dokumentation"
Überarbeitet: August 2002

Zur BEMASSUNG der WINKEL sind drei Punkte erforderlich.

Im Uhrzeigersinn geben Sie nacheinander an:

  1. Ersten Schenkel anklicken
  2. Zweiten Schenkel anklicken
  3. Scheitelpunkt anklicken
  4. Und dann: Position der Maßlinie festlegen


Exakte Winkelbestimmung:

  1. Mit <Strg>+<F9> Konstruktionsfang einschalten
  2. Suchfenster springt auf exakte Konstruktionspunkte


Die Korrektur der Bemaßung ist auf folgende Weise möglich:

  • Zunächst wird über die Icon-Gruppe Dokumentation die normale Bemaßung erstellt.
  • Zur Korrektur rufen Sie in der Icon-Gruppe Bearbeitung das Icon KORREKTUR TEXTE auf und wählen mit dem Suchfenster die Bemaßung, deren Text Sie verändern wollen. Nach dem Klick erscheint ein Fenster KORREKTUR Bemaßung.
  • Ändern Sie unter Neue Länge den Maßeintrag und unter Texthöhe die vorgegebene Höhe.
    • Wenn Sie unter Neue Länge einen Punkt oder die Zahl 0 eingeben, verschwindet der Maßeintrag. Er kann später durch einen beliebigen freien Text ersetzt werden (siehe dazu Beschriftung).
  • Mit OK schließen Sie die Korrektur ab.
