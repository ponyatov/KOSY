Datum letzte Änderung

Icon in der Icon-Gruppe "Dokumentation"
Überarbeitet: August 2002

Das Datum, zum Zeitpunkt der letzten Änderung, wird an die gewählte Position geschrieben. Es entsteht ein spezielles Zeichnungsteil im Layer 9.

Nach einem Klick auf das Icon sind folgende Tätigkeiten erforderlich:

  1. Fadenkreuz positionieren.
  2. Mit einem Mausklick bestätigen - oder durch Klick auf ein anderes Icon die Funktion abbrechen.

Das enstandene Zeichnungsteil kann über verschiedene Bearbeitungs-Funktionen weiterbehandelt werden (Löschen, Verschieben, Kopieren usw.). Eine Textkorrektur darf jedoch nicht zugelassen werden.
