Konfiguration von nccad
Überarbeitet: Februar 2005


Erstellen Sie sich eine individuelle Version von nccad. Sie können:

  • Neue ICON-Leisten erstellen
  • Bestehende Icon-Leisten verändern
  • Den Funktionsumfang verändern
  • Die Menüs reduzieren
  • Die Starteigenschaften beeinflußen

Bitte beachten: Die Konfiguration muß mit der aktuellen Version erstellt - und mit derselben Version benutzt werden. Eine Verwendung der Konfiguration bei anderen Versionen führt zu Fehlermeldungen.

Zum Erstellen einer Konfiguration die folgenden Hinweise:

 


1. Neue ICON-Leiste erstellen und verändern
Wenn Sie mit dem Mauscursor in die Menüzeile fahren und die rechte Maustaste drücken, erscheint das obige PopUp-Menü, mit dessen Hilfe Sie z.B. das ICON-Menü ausblenden können. Wenn Sie Anpassen wählen, können Sie vorhandene ICON-Leisten verändern oder neue erstellen, dazu die folgenden Hinweise:


1.1 Neue Leiste aufrufen
Wenn Sie Anpassen gewählt haben, klicken Sie auf Neu und geben den Namen für die neue ICON-Leiste ein (siehe Bild). Nachdem Sie OK geklickt haben, erscheint an einer beliebigen Position eine leere ICON-Leiste, die Sie an eine andere Position ziehen können.


1.2 Neue Leiste füllen
Im Fenster Anfassen wählen Sie den Reiter Befehle und dort im Feld Kategorien die ICON-Gruppe oder das Menü aus dem Sie ein ICON auswählen - und in der neuen Leiste positionieren wollen. Mit der linken Mautaste wählen Sie das gewünschte ICON und ziehen es bei gedrückter Maustaste in die neue ICON-Leiste. Dort erscheint an der Einfügestelle zunächst ein Balken (siehe Bild), der mit dem Loslassen der Maustaste verschwindet, in diesem Moment wird das neue ICON eingefügt.

 


1.3 Neue Leiste positionieren
Wenn Sie die ICON-Leiste anfassen und bewegen, kann sie in beliebige Positionen gebracht werden. Sobald Sie an markante Kanten fahren wird die Leiste als gestrichelte Linie dargestellt (siehe Bild). Lassen Sie jetzt die Maustaste los, wird die Leiste "angedockt".


1.4 Neue Leiste positioniert
Das Bild zeigt die angedockte ICON-Leiste unterhalb der Menüzeile, ein idealer Platz für individuelle ICONs.

1.5 ICONs entfernen
Dazu muß das Fenster "Anpassen" geöffnet sein (siehe Bild 1.). Dann können Sie in jeder, am Bildschirm sichtbaren, ICON-Leiste mit der linken Maustaste eines der ICONs anfassen und in die freie nccad-Arbeitsfläche ziehen, damit ist es gelöscht.

 


2. Menü konfigurieren
Wie unter 1. muß zunächst in der Menü-Zeile die rechte Maustaste gedrückt - und Anpassen gewählt werden, damit überhaupt Änderungen vorgenommen werden können.
Sobald das Fenster "Anpassen" sichtbar ist, kann in der Menüzeile eines der Menüs gewählt werden (siehe Bild). Wenn Sie im Menüfenster eine der Zeilen mit der linken Maustaste anfassen und in der freie Fläche ziehen, verschwindet es im Menüfenster.

Durch den Button "Zurücksetzten" im Fenster "Anpassen" kann der ursprüngliche Zustand wieder hergestellt werden.

 

3. Konfiguration abspeichern, öffnen und starten
Wie unter 1. und 2. erklärt, haben Sie ICON-Leisten und Menüs individuell gestaltet. Dieser Zustand muß festgehalten werden und jederzeit zur Verfügung stehen. Dazu die folgenden Hinweise:


3.1 Konfiguration abspeichern oder öffnen
Der momentan auf dem Bildschirm vorhandene Konfigurationszustand wird über das Menü Datei unter Konfiguration speichern in eine Datei übernommen, die mit der Endung *.ncc versehen ist.

Über Konfiguration öffnen kann jederzeit eine abgespeicherte Konfiguration aufgerufen - und verwendet werden.

3.2 Konfiguration automatisch starten
Eine Konfiguration kann zusammen mit dem Start von nccad aufgerufen werden, z.B. durch die folgende Windows-Befehlszeile unter Start/Ausführen oder innerhalb einer Desktop-Verknüpfung:

c:\kosyx\nccadx.exe /ul=0/men:c:\kosyx\beginner.ncc

x steht für Versionsnummer, hier ist normalerweise 7 einzusetzen

Zunächst wird nccad selbst aufgerufen, dann folgt nach ...exe ein Leerzeichen, danach die Informationen über die Konfiguration.
ul=0 bedeutet, daß die Konfiguration nach dem Start von nccad jederzeit veränderbar ist.
ul=1 bedeutet, daß die Konfiguration nach dem Start von nccad nicht veränderbar ist.
ul... kann auch weggelassen werden.

3.3 Konfiguration und Bearbeitungsdatei automatisch starten
Eine Konfiguration und eine Bearbeitungsdatei können zusammen mit dem Start von nccad6 aufgerufen werden, z.B. durch die folgende Windows-Befehlszeile unter Start/Ausführen oder innerhalb einer Desktop-Verknüpfung:

c:\kosyx\nccadx.exe /ul=1/men:c:\kosyx\beginner.ncc c:\Fertigung\Dreieck.knc

x steht für Versionasnummer

Zunächst wird nccad selbst aufgerufen, dann folgt nach ...exe ein Leerzeichen, danach Pfad und Dateiname für die Konfiguration und dann wieder ein Leerzeichen gefolgt von Pfad und Dateiname für die Bearbeitungsdatei, hier ein CNC-Programm.
ul=0 bedeutet, daß die Konfiguration nach dem Start von nccad jederzeit veränderbar ist.
ul=1 bedeutet, daß die Konfiguration nach dem Start von nccad nicht veränderbar ist.
ul... kann auch weggelassen werden.

 


4. Konfigurationsdateien und Start-Buttons
Im Hauptverzeichnis von nccad (siehe Bild oben) sind verschiedene Konfigurationsdateien abgelegt hier ein Auszug:

110_Einsteiger.ncc Diese Version ist als ersterSchritt für den reinen Einsteiger gedacht. Das Zeichnen ist mit der konventionellen Art zu vergleichen. Nur LÖSCHEN und KORREKTUR TEXTE stehen als Bearbeitungsfunktionen zur Verfügung. Diese Konfiguration hat einen geringeren Befehlsumfang als 120_Anfaenger.ncc!
120_Anfaenger.ncc Anfänger benötigen für einfache Zeichen- und Bearbeitungsfunktionen nur wenige Befehle. Grundlagen können hiermit geschult werden.
210_Fortgeschrittener.ncc Fortgeschrittene arbeiten mit Funktionen, die den schulischen Alltag abdecken. Die Maschinensteuerung wird ermöglicht.
300_Alles.ncc In dieser Vollversion sind alle Funktionen nutzbar.
   
310_Platinenfraesen.ncc   Die für das Platinenfräsen notwendigen Funktionen (PAD/BAHN GENERIERUNG – OUTLINE GENERIERUNG – POLYGON GENERIERUNG) sind in die Iconleiste integriert und direkt anwählbar. Der Befehlsumfang entspricht der Konfiguration 210_Fortgeschrittener.ncc.
   
   
   

Bei der Installation von nccad werden Start-Buttons auf dem Bildschirm erzeugt, die eine Konfigurationsdatei verwenden. Das folgende Bild zeigt den Zusammenhang und die Bearbeitungs-Möglichkeiten, die auf folgende Weise erreicht werden:

  1. Klick mit rechter Maustaste auf den Start-Button.
  2. Im PullDown-Menü Eigenschaften wählen.
  3. Unter dem Reiter Verknüpfung erscheint das Feld "Ziel", in dem der Aufruf von nccad unter Berücksichtigung der Konfigurationsdatei (*.ncc) eingetragen ist. Der Eintrag kann verändert werden.
  4. Klicken Sie auf "Anderes Symbol", kann auch das Bild des Buttons verändert werden.
