Gravuren mit fortlaufenden Zahlen
Überarbeitet: August 2002


1. Die Aufgabe
Wenn Sie beispielsweise eine Serie von Typenschildern oder Nummernschilder für einen Wegweiser machen wollen, müssen mit jedem, zu gravierenden Schild, die Zahlen automatisch verändert werden. Dazu verfügt nccad über eine wirkungsvolle Funktion, deren Bedienung im Folgenden erklärt werden soll:

 


2. Die Programmierung
Öffnen Sie im Menü Datei/CAD/CAM - Neue Zeichnung eine Zeichenfläche und wählen Sie eines der ICONs für Gravurtexte, z.B. GRAVURTEXT max/ez , wie im Bild dargestellt. Um einen fortlaufenden Zähler zu programmieren, muß die Zahl von den Zeichenkombinationen */ und /* regelrecht "eingerahmt" werden. Dabei gibt es eine Reihe von Formatierungs-Möglichkeiten:

*//* keine Formatierung der dazwischen zu gravierenden Zahl
*/00/* 2 Stellen werden graviert, mit führender Null
*/000/* 3 Stellen werden graviert, mit führenden Nullen
*/000.00/* 3 Stellen vor- 2 Stellen hinter dem Komma werden graviert, mit führenden Nullen
   
   

Wenn Sie mit OK die Eingabe abschließen und dem Gravurtext eine Technologie zuordnen, können Sie die Werkstück-Bearbeitung vorbereiten und die Zahlenwerte übergeben, dazu der folgende Abschnitt:

 

 


3. Die Zahlenwerte übergeben
Über das Menü Parameter/CNC-Maschine/Besonderes erreichen Sie das gezeigte Fenster, in dem Sie den Reiter Zähler aufrufen. Dabei bedeuten:

  • Aktuell
    Der momentan erreichte Zählerstand (zu Beginn einer Serienproduktion ist der Wert 0).
  • Startwert
    Die erste zu gravierende Zahl, z.B. 1 bei Wegweisern oder 1000 als Anfangswert eines Typenschildes. Über diesen Anfangswert kann auch eine unterbrochene Produktion bei einem aktuellen Zählerstand fortgesetzt werden.
  • Schrittweite
    Ist eine Zahl graviert wird die Schrittweite addiert, um die nächste Zahl zu gravieren, also z.B. 1 für eine fortlaufende lückenlose Schilderserie.

Eine Schilderserie kann besonders rationell mit der Funktion "Werkstück-Kopien" produziert werden, die im selben Fenster eingestellt werden.
