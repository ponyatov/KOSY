Technologien zum Bearbeiten
Überarbeitet: Oktober 2004

Nachdem Sie mit dem CAD-Modul von nccad Konturen (Zeichnungsteile) gezeichnet haben oder die Konturen von anderen CAD-Programmen importiert haben, brauchen Sie weitere Angaben für die Fertigung der Teile - die sog. Technologie-Angaben. Es muss beispielsweise angegeben werden, wie schnell, wie tief, mit welcher Zustellung und auf welche Weise die gezeichnete Kontur bearbeitet werden soll. Dies alles ist unter dem Begriff CAM zusammengefasst (Computer Aided Manufactoring = computerunterstützte Fertigung).
Alle dafür erforderlichen Software-Funktionen sind innerhalb von nccad im CAM-Modul zusammengefasst, das im Laufe der Jahre einen erheblichen Umfang angenommen hat und über sehr komplexe Funktionen verfügt.
Im Folgenden soll zunächst im 1. Kapitel das Prinzip erklärt werden, die einzelnen Technologien sind im Anhang gesondert zusammengefasst.

 


1. Arbeiten mit Technologieen
1.1 Zeichnen und Technologie aufrufen
Zuerst wird das Zeichnungsteil gezeichnet. Dazu wählen Sie im Menü Datei/ CAD/CAM..... - Neue Zeichnung und dann im Icon-Menü in der Gruppe CAD Standard das Icon Gerade. Mit zwei Mausklicks geben Sie die Endpunkte der Geraden ein. Dann klicken Sie auf das Icon TECHNOLOGIE in der Gruppe CAM, es wird das Fadenkreuz mit einem Suchfenster aktiviert. Das Zentrum dieses Suchfensters bewegen Sie zu einem der Konstruktionspunkte der Geraden und warten kurze Zeit. Es "rastet" am Konstruktionspunkt ein, in der Statuszeile (ganz unten, unterhalb der Zeichenfläche) wird gemeldet, dass eine Gerade gefunden wurde, gleichzeitig wird die Gerade farblich markiert. Drücken Sie unter dieser Bedingung die linke Maustaste (klicken) und bewegen Sie die Maus etwas: An der Geraden hängt ein "Gummifaden" - die sogenannte Zuordnungslinie. Jetzt wird nochmals die linke Maustaste gedrückt. Es erscheint das folgende Technologie-Editierfenster. 

 


 1.2 Editieren der Technologie
Im Technologie-Fenster wird in Zeile 1 die Bearbeitungs-Nummer eingegeben. In den weiteren Zeilen folgen die Technologie-Angaben für die gewünschte Bearbeitung, wie im Bild beispielhaft dargestellt.
Durch einen Klick auf OK verlassen Sie das Fenster, es erscheint beispielsweise das folgende Bild.

 


 1.3 Kontur mit Technologie
Die Gerade wird in ihrer ursprünglichen Form dargestellt, am Ende "hängt" die rote Zuordnungslinie mit der Angabe der Nummer - und, nach einem Schrägstrich, die Layer-Nummer, falls sie nicht 1 ist.
Informationen in rot werden übrigens immer der Zeichnungslage (Layer) 10 zugeordnet.
Sollten mehrere Konturen gezeichnet werden, erhält jede Kontur auf die gleiche Weise eine Technologie. Die Nummern bestimmen die Reihenfolge der Bearbeitung, sie dürfen deshalb nur einmal vergeben werden.

 


1.4 Ändern der Technologie
Dazu ist folgendes zu tun: Das Icon TECHNOLOGIE klicken, Suchfenster an das Ende der roten Zuordnungs-Linie positionieren, Markierung abwarten und klicken. Es erscheint wiederum das Technologie-Fenster, in dem Sie die Angaben überschreiben können. Auf OK klicken, damit sind die geänderten Technologie-Angaben übernommen. Vergessen Sie nicht, die Datei unter gleichem Namen (Speichern) oder unter einem neuen Namen (Speichern unter) abzuspeichern!

 


1.5 Technologien für verschiedene Konturen
Besonders leistungsfähig wird nccad dadurch, dass jedem Zeichnungsteil eine individuelle Technologie zugeordnet werden kann, d.h. jedes der gezeichneten Teile kann unterschiedlich bearbeitet werden. Dazu erhält es über eine "Zuordnungslinie" und über die Nummerierung seine spezifischen Technologiedaten, das obige Beispiel zeigt das Prinzip.
Eine Gleichbehandlung aller Teile ist ebenfalls möglich, indem nur einem der Teile eine Technologie mit der Nummer 0 zugeordnet wird. 
Sollen individuelle- und gleiche Bearbeitung gemischt werden, so ist auch dies über die Nummernvergabe erreichbar: Die individuell zu bearbeitenden Teile erhalten Nummern ab 1, eines der gleich zu bearbeitenden Teile erhält die Nummer 0. 

Zwischen den Nummern können Lücken gelassen werden.

 


 1.6 Technologien in verschiedenen Layern
Oft muß die Bearbeitungsreihenfolge sehr komplex gestaltet werden. Um einen besseren Überblick zu erhalten, können die Zeichnungsteile auf mehrere Layer (Zeichnungslagen) verteilt werden. Zur Technologie-Nummer wird automatisch die Layer-Nummer ergänzt (z.B.: 10/4), wenn der Layer größer 1 ist. 
Die Nummer 0 kann in allen Layern vorhanden sein. Die Nummern größer 0 dürfen allerdings nur einmal vergeben werden. 
Die Abarbeitung des obigen Beispiels geschieht in folgender Reihenfolge:
1. Die Nummern 1, 2, usw. im Layer 1
2. Die Nummern 3 höheren Layer
3. Das Teil mit der Nr. 0 im Layer 1
4. Die Teile ohne Technologie im Layer 1, bearbeitet wie Nr. 0

Über das Icon Layer in der Gruppe Darstellung kann die zugehörige Bearbeitung völlig umgangen werden, indem dieser Layer unsichtbar gemacht wird.

 

2. Erweiterte Technologieen
Eine Fülle von weiteren Bearbeitungsfunktionen, Strategieen und Hilfsfunktionen erschließen sich über die verschiedenen Reiter am oberen Rand desTechnologie-Fensters. Die Wichtigsten davon sollen im Folgenden erklärt werden, soweit sie nicht selbsterklärend und trivial sind.


2.1 Reiter "Steuerung"
Dazu folgende Bemerkungen:

  • Fräsrichtung
    Im Zusammenhang mit Material und Fräserkonstruktion kann sie eine Rolle spielen, man spricht hier auch von Gegenlauf- und Mitlauf-Fräsen. Die Einstellung hat nur Wirkung bei geschlossenen Zeichnungsteilen, wie Kreisen, Taschen u.Ä., nicht aber....
    .... bei Strecken an denen die Bearbeitung dort beginnt, wo die Technologie-Zuordnungslinie "hängt",
    .... bei Anwendung der Technologie-Nummer 0.
  • Spannung
    Sofern die Elektronik über einen Analog-Ausgang verfügt (ab 2004 bei allen gelieferten Maschinen) kann an dieser Stelle der Wert der Spannung zwischen 3 und 25 Volt.
  • Pause...
    Vor- oder nach der Ausführung der im Fenster genannten Technologie kann eine Wartezeit programmiert werden. Besondere Bedeutung hat dies bei Verwendung einer Frässpindel mit Hochlaufzeit.
  • Digitale Eingänge
    Für Automatisierungs-Anwendungen. Mehr dazu unter Automation.

 


2.2 Reiter "Eintauchen"
Bei vielen Materialien ist ein sanftes Eintauchen in das Material erforderlich, es schont Maschine und Werkzeug und erzeugt weniger Bearbeitungsgeräusche. Genutzt wird hier eine, ab nccad 7 zur Verfügung stehende, Funktion für Wendelbewegungen (Helix).

Sie muß lediglich aktiviert werden (siehe Bild oben). Beim Eintauchen wird dann ein Wendel bis zur Zustellungseben gefahren, wie das Bild unten in der Simulation zeigt.
Bitte beachten Sie, daß dieses Helix-Eintauchen nur möglich ist, wenn...

..... genug Platz für eine solche Bewegung vorhanden ist, also der Fräser wesentlich kleiner ist als das entstehende Loch.

..... die Zeichnungsteile Sackloch, Stufenbohrung, Tasche bearbeitet werden sollen.

Der Überlappungsfaktor hat in der Normalversion keine Auswirkungen.


2.3 Reiter "Mindermengen-Dosierung"
Hier muß die optional lieferbare Mindermengen-Dosierung vorhanden sein. Sie finden Hinweise dazu unter Optionen/Mindermengen-Dosierung.

2.4 Reiter "Punkt-/Bahn-Dosierung"
Hier muß das optional lieferbare KOSY-Dosiersystem vorhanden sein. Sie finden Hinweise dazu unter Grundlagen Dosiersystem.

 

 


3. Technologie-Vorgaben
Da die Anzahl der Technologie-Angaben mittlerweile so umfangreich ist, werden sie auf 2 Seiten verteilt, zwischen denen über das Feld Seite 2 oder Seite 1 unten im Technologie-Fenster umgeschaltet werden kann. Die wichtigsten Angaben sind jedoch im Hauptfenster (Seite 1) zusammengefasst.

Damit häufig wiederkehrende Technologien nicht immer wieder eingegeben werden müssen, können Vorgaben individuell eingestellt werden. Dazu gehen Sie in das Menü Parameter/Technologie, es erscheint das obige Fenster mit den Technologie-Seiten, in denen Sie - soweit sinnvoll - Ihre Vorgaben editieren können. Wenn Sie auf Speichern klicken, werden diese Vorgaben dauerhaft zur Verfügung stehen.

Einen Überblick über die vorhandenen Technologie-Angaben- und deren Einsatz finden Sie im Anhang.

 


4. Bearbeitungs-Datenbanken
Bearbeitungsdaten - wie Vorschub, Zustellung, Spindeldrehzahl - können auch aus der mitgelieferten Bearbeitungs-Datenbank entnommen werden. Dort hinterlegte Daten sind immer unter Vorbehalt zu übernehmen: Sie wissen nicht immer exakt das Material (ist es z.B.: AlMg3 oder AlMgCuPb) und es kann eine falsche Datenbank verwendet werden (durch Fremde geänderte Dateien oder Einträge). 

Um die Bearbeitungsdatenbank aufzurufen klicken Sie im Technologie-Hauptfenster auf das Feld Bearbeitungsdatenbank, wählen das entsprechende Material, die Bearbeitungsart, den Werkzeugtyp und die Vorschläge in den Feldern 1 bis 4. Die gefundenen Werte werden automatisch in die Felder des Technologiefensters eingetragen, wenn sie auf Daten übernehmen klicken. Die Einträge können im Technologie-Fenster aber auch nachträglich editiert werden.

Besonderen Technologien für Ihre ganz individuellen Materialen ermitteln Sie in Grundlagenversuchen. Diese Einstellungen übernehmen Sie dadurch in eine persönliche Datenbank, indem Sie im Technologie-Fenster auf Privatdaten bearbeiten klicken.
